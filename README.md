# OWASP Security POC

This project is a deliberately vulnerable Node.js application created to test AI-powered security code review tools such as GitHub Copilot.

## Objective
Identify OWASP Top 10 vulnerabilities using AI models.

## How to Run

1. Install dependencies:
   npm install

2. Start MySQL and execute the database script.

3. Run the application:
   npm start

4. Access the app:
   http://localhost:3000

## Warning
This project contains intentional security vulnerabilities. Do NOT deploy in production.
